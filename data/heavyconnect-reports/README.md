# HeavyConnect report evidence import

This folder contains a structured import from PDFs downloaded out of HeavyConnect for the date range January 1, 2025 through January 1, 2026.

The original PDFs remain in `C:/Users/kenneth/Desktop/heavyconnectreports/` and are intentionally not copied into git. `inventory.json` records metadata, report type grouping, bounded sample fields, evidence tags, and a first-pass mapping to PrimusGFS v4.0 modules.

All seven report types previously identified as important for Bay Baby are represented in the import.
